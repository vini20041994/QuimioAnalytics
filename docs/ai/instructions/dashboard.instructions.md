# Regras de Dashboard

Referência documental da instrução ativa em `.github/instructions/dashboard.instructions.md`.

- Destaque primeiro os KPIs mais relevantes para a decisão.
- Sempre mostre o contexto da métrica: unidade, escopo do filtro e base temporal ou amostral.
- Agrupe visuais relacionados para que o fluxo de leitura favoreça comparação e diagnóstico.
- Evite poluição visual que compita com a mensagem analítica.
- Mantenha filtros fáceis de encontrar e resultados simples de interpretar.
