# Regras de UX

Referência documental da instrução ativa em `.github/instructions/ux.instructions.md`.

- Otimize a interface para leitura analítica rápida, não para complexidade decorativa.
- Mantenha rótulos, estados vazios e mensagens de erro operacionalmente claros.
- Garanta que filtros e tabelas permaneçam compreensíveis sob alta densidade de dados.
- Use hierarquia visual para guiar a atenção primeiro aos resultados críticos.
- Evite padrões que façam o produto parecer um chatbot ou landing page.
