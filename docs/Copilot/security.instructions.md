## Diretrizes de Segurança

Sempre:

- validar entradas;
- sanitizar dados;
- proteger endpoints;
- utilizar RBAC;
- implementar autenticação segura;
- utilizar variáveis de ambiente;
- evitar exposição de dados sensíveis;
- registrar ações críticas.

Nunca:

- expor secrets;
- armazenar senhas em texto puro;
- confiar no frontend;
- utilizar queries inseguras.
