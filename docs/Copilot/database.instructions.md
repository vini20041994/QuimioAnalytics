## Banco de Dados

Sempre:

- modelar para rastreabilidade;
- utilizar chaves adequadas;
- normalizar quando necessário;
- desnormalizar apenas quando justificado;
- criar índices;
- evitar duplicidade;
- registrar auditoria;
- manter histórico de alterações;
- priorizar integridade referencial.

Estruturas importantes:

- tabelas de auditoria;
- versionamento de registros;
- trilha de alterações;
- controle de importações;
- controle de processamento;
- status de pipeline.
