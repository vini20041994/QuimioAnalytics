## Backend

Sempre gerar:

- tratamento completo de erros;
- validações robustas;
- logs estruturados;
- mensagens claras;
- paginação;
- filtros;
- ordenação;
- controle de concorrência;
- proteção contra dados inconsistentes;
- testes automatizados;
- documentação automática.

Evitar:

- queries N+1;
- lógica duplicada;
- funções gigantes;
- controllers extensos;
- queries sem índices;
- código sem tipagem;
- hardcode.
