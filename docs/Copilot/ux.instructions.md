## Diretrizes de UX/UI

O sistema deve ter:

- visual profissional;
- foco analítico;
- alta legibilidade;
- hierarquia visual clara;
- visualização científica;
- experiência fluida;
- baixa poluição visual.

Priorizar:

- tabelas avançadas;
- gráficos analíticos;
- filtros inteligentes;
- dashboards executivos;
- feedback visual;
- estados claros.
