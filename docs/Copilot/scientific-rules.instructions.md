## Sistemas Científicos

O sistema deve respeitar:

- rigor científico;
- rastreabilidade experimental;
- preservação dos dados originais;
- reprodutibilidade;
- consistência analítica;
- integridade dos cálculos;
- auditabilidade.

Nunca:

- sobrescrever dados brutos;
- perder metadados;
- ocultar transformações;
- simplificar cálculos científicos sem justificativa;
- remover precisão relevante.

Sempre:

- manter dados raw;
- versionar processamento;
- registrar parâmetros analíticos;
- registrar origem dos resultados;
- registrar algoritmos utilizados;
- registrar versão do pipeline.
